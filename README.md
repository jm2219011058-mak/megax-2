# Mega X Holding Ltd. — Corporate Website

A professional, deployment-ready corporate website for Mega X Holding Ltd., a global semiconductor holding company specializing in full-band wireless connectivity chipsets, edge-AI, and hardware security.

## Pages

- **index.html** — Home / landing page with hero, product portfolio, market data, roadmap, and competitive advantages
- **about.html** — Company overview, mission, leadership team, values, and careers CTA
- **contact.html** — Contact form, office locations, and department-specific contact details
- **404.html** — Custom error page

## Tech Stack

- Vanilla HTML5, CSS3, JavaScript (no build tools required)
- Google Fonts: Exo 2, Outfit, Share Tech Mono
- CSS custom properties for theming
- IntersectionObserver for scroll-triggered animations
- Fully responsive (mobile-first, breakpoints at 480px, 768px, 1024px)

## Quick Start

Open `index.html` in any browser, or run a local server:

```bash
python3 -m http.server 8000
```

See `DEPLOY.md` for full deployment instructions.
